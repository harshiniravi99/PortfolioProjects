CAR SALES PROJECT: Process and visualize car sales data using Excel and IBM Cognos Analytics to provide insights on sales and profits for each dealer. This included finding and creating visualizations for various KPI metrics such as finding profits by the manufacturing year, quanity sold per dealer, sum of profits, etc.


COVID PROJECT:  I wanted to practice working with large data sets that would have various fields to work with and several angles at which the data could be analyzed and visualized. I also wanted to use this project as an opportunity to get started in learning Tableau and using it to create a dashboard to visualize my analysis.
